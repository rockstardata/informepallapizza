# Informe Pallapizza

Este proyecto es una aplicación web desarrollada con React, Vite y Tailwind CSS para la gestión y visualización de informes de Pallapizza.

## Características principales
- Interfaz moderna y responsiva con Tailwind CSS
- Estructura optimizada con Vite para desarrollo rápido
- Código modular en React

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/rockstardata/informe_pallapizza.git
   cd informe_pallapizza
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Scripts disponibles
- `npm run dev`: Inicia el servidor de desarrollo
- `npm run build`: Genera la versión de producción
- `npm run preview`: Previsualiza la app en modo producción
